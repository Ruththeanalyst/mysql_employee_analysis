# MySQL Employee Analysis
According to the analysis performed on the `parks_and_recreation` dataset, this project explores employee demographics and salary distribution using MySQL.

## Objective
The objective of this analysis is to:
- Explore employee demographic information
- Analyze salary patterns across employees
- Apply SQL filtering, aggregation, and ranking techniques
- Demonstrate analytical thinking using SQL

## Analysis Breakdown
- **Exploratory Analysis:** SELECT, DISTINCT
- **Filtering Logic:** WHERE, LIKE, AND / OR
- **Aggregations:** GROUP BY, HAVING
- **Advanced Analysis:** CTEs, Subqueries, Window Functions
- **Text Processing:** String functions

## Tools Used
- MySQL
- SQL

## Dataset
Sample dataset: `parks_and_recreation`

## Project Structure
Each SQL file focuses on a specific analytical concept, progressing from basic to advanced.
